01 WiFi

wpa_supplicant — WiFi İstemci

wpa_supplicant.conf yaz, WPA2-PSK ve WPA3-SAE ile ağa bağlan. 802.1X/EAP kurumsal kimlik doğrulama, wpa_cli ile interaktif kontrol, roaming ve systemd servis entegrasyonu.

9 bölüm wpa_supplicant · wpa_cli · WPA3 · EAP

02 WiFi

hostapd — Access Point Modu

Linux'u WiFi access point'e dönüştür. Bridge modu, dnsmasq DHCP entegrasyonu, WPA3/SAE, gizli SSID, MAC filtresi ve HTCapab ile bant genişliği kontrolü. Embedded router senaryosu.

9 bölüm hostapd · dnsmasq · bridge · WPA3
Rehbere git
03 WiFi

iw — nl80211 Wireless Arayüzü

iwconfig'in modern halefi iw ile WiFi interface yönetimi. Tarama, sinyal kalitesi, kanal ve txpower ayarı. Monitor mode, mesh mode, rfkill ve regulatory domain. Kernel CFG80211 config.

8 bölüm iw · nl80211 · rfkill · CFG80211
Rehbere git
04 Bluetooth

BlueZ — Linux Bluetooth Yığını

BlueZ D-Bus mimarisi, bluetoothctl ile cihaz tarama/eşleştirme/bağlama. hciconfig ve hcitool araçları. Python pydbus ile D-Bus API. systemd bluetooth.service ve embedded minimal config.

9 bölüm BlueZ · bluetoothctl · D-Bus · pydbus
Rehbere git
05 Bluetooth

BLE GATT — Servis Yazma

BLE GATT hiyerarşisi: Profile → Service → Characteristic → Descriptor. BlueZ D-Bus GATT server API ile Python'da özel servis yaz. UUID tanımı, NOTIFY/INDICATE, MTU, Yocto entegrasyonu.

9 bölüm BLE · GATT · BlueZ D-Bus · Python · Yocto
